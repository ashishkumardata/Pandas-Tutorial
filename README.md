# 📊 Pandas Practice Notebooks

This repository contains my hands-on learning and practice notebooks for **Pandas**, a powerful Python library for data analysis and manipulation.

---

## 🚀 What is Covered

### 📌 Core Concepts

* **Series in Pandas**

  * Creating and manipulating Series
  * Indexing and basic operations

* **DataFrames in Pandas**

  * Creating DataFrames
  * Understanding rows, columns, and structure

---

### 📥 Data Handling

* **Importing Files**

  * Reading data from CSV and JSON files
  * Working with real datasets (`data.csv`, `data.json`)

---

### 🔍 Data Selection & Filtering

* **Selection**

  * Selecting rows and columns using `.loc[]` and `.iloc[]`

* **Filtering**

  * Conditional filtering
  * Boolean indexing for data extraction

---

### 📊 Data Analysis

* **Aggregation**

  * `sum()`, `mean()`, `count()`, `groupby()`
  * Performing summary statistics

---

### 🧹 Data Cleaning

* Handling missing values
* Removing duplicates
* Data type conversion
* Preparing data for analysis

---
